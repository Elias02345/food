# MacroKitchen 1.0.0

Initial production release of the local-first protein recipe PWA.

## Included

1. 225 whole-food recipes with 45 recipes in each meal category
2. Reproducible nutrition calculations from ingredient reference values
3. Macro matching with automatic portion scaling and two-recipe combinations
4. Recipe browsing, filters, favorites, weekly planning, pantry and shopping list
5. Guided cooking with timers, ingredient checklist and screen wake lock
6. PWA installation, offline cache, responsive layout and local JSON backups
7. Docker, automated quality checks and GitHub Pages deployment

## Validation

The release was validated with ESLint, seven automated tests and a production Vite build before publication.
